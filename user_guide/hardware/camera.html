Cameras

The software supports camera-based acquisition. It can run both normal and rolling shutter modes of contemporary scientific CMOS cameras.

Hamamatsu

Note

navigate has been tested with the following versions of the Hamamatsu’s drivers:

  • DCAM API: 20.7.641, 21.7.4321, 22.9.6509, 22.11.4321, 23.12.6736

  • Camera Firmware: 2.21B, 2.53.A, 3.20.A, 4.30.B,

  • Active Silicon CoaXpress: 1.10, 1.13, 1.21.

ORCA-Flash4.0 V3

  • Install Directions.

  • ORCA-Flash 4.0 v3 Manual.

Configuration File
microscopes:
  microscope_name:
      camera:
        hardware:
          type: HamamatsuOrca
          serial_number: 111
          camera_connection:
        defect_correct_mode: 2.0
        delay: 1.0  #ms
        settle_down: 0.1 #ms
        flip_x: False
        flip_y: False

Synthetic Camera

The synthetic camera simulates noise images from an sCMOS camera. If no camera is present, the synthetic camera class must be used.

Configuration File
microscopes:
 microscope_name:
     camera:
       hardware:
         type: synthetic
         serial_number: 111
         camera_connection:
       defect_correct_mode: 2.0
       delay: 1.0  #ms
       settle_down: 0.1 #ms
       flip_x: False
       flip_y: False
